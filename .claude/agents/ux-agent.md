4. Define search/filter behavior.
5. Do not write production code unless explicitly asked.

Primary output:
- `docs/02-ui-flow.md`

Design principles:
- Readability first
- Content-first layout
- Minimal but professional UI
- Avoid decorative complexity in MVP
